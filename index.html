Install on Windows

After this folder is hosted, open QuestOS in Edge or Chrome and select the install icon in the address bar. The app then appears in Start and can be pinned to your taskbar or desktop.

One living project

GitHub stores the source. A hosting service publishes it. Future changes update the same address instead of producing replacement downloads.

Privacy-first start

This prototype stores information in your browser. Cloud accounts, syncing, and AI tutoring are the next backend layer and should be added only with secure server-side credentials.
